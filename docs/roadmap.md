## 4. v3.0 — 智能记账与深度洞察 (短期规划)

**代号**: "Smart"
**目标**: 降低记账成本, 提升数据价值

### 4.1 周期性交易
- 支持设置重复规则: 每日/每周/每月/每年
- 到日期自动创建交易记录
- 典型场景: 工资、房租、订阅服务
- 预估: 后端 + 前端, 1 周

### 4.2 报表系统
- 月度/季度 PDF 报表
- 数据: 收入趋势、支出分类、环比对比
- 支持发送到邮箱
- 预估: 后端 (PDF 生成) + 前端, 1.5 周

### 4.3 拍照记账
- 上传小票照片
- OCR 识别金额 + 日期 + 商家
- 智能匹配分类 (基于历史)
- 预估: 接入第三方 OCR API, 2 周

### 4.4 支出预警
- 自定义分类月预算
- 当支出达到阈值时预警 (80%/100%)
- 简单通知 (站内消息)
- 预估: 后端 + 前端, 1 周

### 4.5 PWA 移动适配
- Service Worker 离线支持
- 移动端布局优化
- 主屏幕添加到桌面
- 预估: 前端, 1 周

### 4.6 汇率自动更新
- 接入免费汇率 API (ExchangeRate-API / 国家外汇管理局)
- 每日自动拉取最新汇率
- 预估: 后端 (定时任务), 0.5 周

## 5. v4.0 — 生态扩展 (远期展望)

**代号**: "Ecosystem"
**目标**: 构建个人财务数据枢纽

### 5.1 银行流水自动导入
- 对接 Plaid (海外) / 网银 CSV 解析
- 自动分类匹配
- 定期自动同步

### 5.2 账本共享
- 创建家庭账本
- 邀请成员共同记账
- 权限控制: 管理员 / 编辑者 / 查看者

### 5.3 自然语言记账
- 输入 "今天吃饭花了 35" 自动记账
- AI 分类建议
- LLM 驱动的智能查询 ("上个月交通花了多少")

### 5.4 年度财务报告
- 全年收支总览
- 消费习惯分析
- 储蓄率统计
- 可导出 PDF

### 5.5 预算管理
- 按月/按分类设置预算
- 预算 vs 实际对比图
- 超预算提醒
